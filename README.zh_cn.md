# Boat

### 依赖

必要:

- [cmake](http://www.cmake.org/) - build system
- [serial](http://wjwwood.github.com/serial/) Cross-platform, Serial Port library written in C++ 

### 编译

Get:

`git clone https://github.com/wjwwood/serial.git`

build:

`mkdir build
cd build
cmake ..
make`

### 配置文件
配置文件在boat下config.xml

参数说明：

- gps 自主航行预置航线

### 运行

在boat目录下执行
`sudo build/./Ship`
